Vim Browser Help

Navigation

j/kSmooth scroll down/up
d/uSmooth page down/up
g/GTop/bottom of page
H/LBack/forward
rReload page

Buffers (Tabs)

n/pNext/previous buffer (with fade transition)
tNew buffer
xClose buffer
:bShow buffers
:1,2,3...Go to buffer number

Opening URLs & Search

oOpen URL (with Tab completion for history)
eOpen current page in external browser
sSmart search (Google or URL)
aAI search (ChatGPT)
SpaceNative AI chat
/Search in page

Command Prompt Features

Ctrl+Up/DownCycle between command modes (:, /, o, s, a)
TabComplete commands (:) or cycle URL history (o)
Ctrl+SpaceShow recent URL history (in o mode)

Developer Tools

F10Toggle developer tools
Ctrl+UView page source
Ctrl+IShow debug info
Ctrl+Shift+OOpen current page in external browser

AI Integration

SpaceAsk AI anything - it can navigate or create content
(shows loading overlay with spinner)
Examples:
"navigate to github" → Opens GitHub
"create a todo list" → Generates interactive todo app
"make a calculator" → Creates working calculator
"explain quantum physics" → Generates explanation page

Commands (:)

:qQuit
:helpShow this help
:e <url>Open URL
:bdClose buffer
:bn/:bpNext/previous buffer
:export-htmlExport page as HTML snapshot
:export-mdExport page as Markdown
:export-pdfExport page as PDF
:browserOpen current page in external browser
:browser <name>Open in specific browser (firefox, chrome, etc.)
:browser-listList available browsers
:extOpen current page in external browser (shorthand)

Modes

NORMALDefault mode for navigation
COMMANDTyping commands or URLs
EscapeReturn to NORMAL mode

Examples

s python tutorial → Google search

s github.com → Open GitHub

a explain quantum computing → Ask ChatGPT

Space navigate to reddit → AI opens Reddit

Space create a calculator → AI generates calculator

o reddit.com → Open Reddit

Press Escape to return to normal browsing
